# MyTaskList App

**Nama:** Samuel Ricardo Mendrofa

**NIM:** 243303621285

## Deskripsi

MyTaskList App adalah aplikasi manajemen tugas (to-do list) sederhana berbasis React Native menggunakan Expo. Aplikasi ini dirancang untuk membantu pengguna mencatat, mengatur, dan memantau tugas harian dengan tampilan modern, responsif, serta mudah digunakan.

Aplikasi ini memiliki fitur utama seperti menambahkan tugas baru, mengedit tugas yang sudah ada, menghapus tugas dengan konfirmasi, serta menandai tugas yang telah selesai. Setiap tugas juga disertai informasi tanggal dan jam saat dibuat sehingga pengguna dapat mengetahui waktu pencatatan tugas.

Selain itu, tersedia fitur pencarian tugas secara real-time, filter daftar tugas berdasarkan status (All, Active, Done), progress bar penyelesaian tugas, serta counter jumlah tugas selesai dari total tugas. Aplikasi juga mendukung Dark Mode dan Light Mode untuk memberikan kenyamanan penggunaan sesuai preferensi pengguna.

Untuk meningkatkan pengalaman pengguna, aplikasi dilengkapi animasi saat menambah, menghapus, atau mengubah status tugas, serta tampilan Empty State yang menarik ketika belum ada tugas yang tersedia.

## Fitur Utama

* Tambah tugas baru
* Edit tugas
* Hapus tugas dengan konfirmasi
* Tandai tugas selesai / belum selesai
* Tanggal & jam pembuatan tugas
* Search task secara real-time
* Filter All / Active / Done
* Statistik progress tugas
* Progress bar penyelesaian
* Dark Mode / Light Mode
* Animasi interaksi task
* Empty state modern


## Cara Menjalankan

```bash
npm install
npx expo start
```
