## 📌 Deskripsi

Aplikasi task manager sederhana berbasis React Native + Expo yang dapat menambah, menghapus, menandai selesai, mencari task, filter task, dark/light mode, dan progress statistik.

## ✅ Fitur

* Add Task
* Delete Task
* Mark as Done
* Search Task
* Filter All / Active / Done
* Progress Bar
* Dark / Light Mode
* Empty State
* FlatList Dynamic Data

## ▶️ Cara Menjalankan

```bash
npm install
npx expo start
```
